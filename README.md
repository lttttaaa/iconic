# Iconic - Obsidian 图标管理插件

Iconic 是一款用于 Obsidian 的图标管理插件。

它可以帮助用户收藏常用图标，并通过网格方式快速插入笔记。

## 功能特点

- ✅ 图标收藏管理
- ✅ 支持快速搜索图标
- ✅ 支持自定义收藏列表
- ✅ 网格化展示图标
- ✅ 一键复制图标
- ✅ 适合笔记、知识库、任务管理使用

## 安装方法

### 方法1：手动安装

1. 下载插件文件
2. 解压到：你的Vault/.obsidian/plugins/iconic
